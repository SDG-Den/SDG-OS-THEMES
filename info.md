Package Name: sdg-themes
Descriptive Name: SDG OS Wallpaper Themes
Source: https://git.sdgcloud.nl/SDGDen/SDG-OS-THEMES
Maintainer: SDGDen <sdgden@sdgcloud.nl>
Version:0.2

Dependencies: 
DMS (DankMaterialShell), mmsg, fzf, notify-send
Optional: Matugen

Description: 
Curated collection of ~150+ wallpaper images organized into 56 named wallpaper groups. Each group has a wallpaper.conf that controls DMS theming (Matugen variant, dark/light mode, DMS preset). Wallpaper groups are categorized as dynamic (auto-extract colors), registry (DMS preset), custom (hardware-brand), or generic (fixed color).
