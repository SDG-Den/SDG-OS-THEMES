Package Name: sdg-themes
Descriptive Name: SDG OS Wallpaper Themes
Source: https://github.com/SDG-Den/SDG-OS-THEMES
Maintainer: SDGDen
Version:0.5.1

Dependencies: 
DMS (DankMaterialShell), mmsg, fzf, notify-send
Optional: Matugen

Description: 
Curated collection of wallpapers organized into named wallpaper groups. Each group has a wallpaper.conf that controls DMS theming (Matugen variant, dark/light mode, DMS preset). Wallpaper groups are categorized as dynamic (auto-extract colors), registry (DMS preset), custom (hardware-brand), or generic (fixed color).
