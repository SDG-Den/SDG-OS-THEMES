# SDG-OS-THEMES Tips

## SUPER+W — Open wallpaper group selector
Launches the wallpaper group FZF picker bound to `setwallpapergroup.sh`.

## ALT+W / ALT+SHIFT+W — Cycle wallpapers
Cycle forward/backward through wallpapers in the current group. Handled by DMS.

## wallpaper-select
Run `wallpaper-select <group-name>` to set a group directly without the FZF picker.

## Per-group color schemes
Each group can define a `wallpaper.conf` with Matugen palette, Material You color,
dark/light mode, and DMS theme preset. See `docs/` for details.
