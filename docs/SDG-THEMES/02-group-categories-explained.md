# Group Categories Explained

Each wallpaper group has a `Theme_Category` field that determines how colors are applied.

## dynamic (20 groups)

Colors are extracted automatically from the wallpaper using Matugen (Material You). The `Generic_Color` field is set to `dynamic`, and no preset is used.

Groups: anime, default, destiny-arc, destiny-class, destiny-solar, destiny-void, ethereal, gaming, ganjaos, hackerman, jade, landscapes, lumon, matey, pride, risetto, snow, solitude, vantablack, yuri

## registry (19 groups)

A predefined DMS theme preset is applied (e.g., nord, catppuccin, gruvbox). The `Generic_Color` is set to `custom` and `Preset` names a DMS theme file.

Groups: AMOLED, arch, catppuccin, catppuccin-latte, everforest, flexoki, great-wave, great-wave-dark, gruvbox, modus, nord, retro-82, rose-pine, steamdeck, terminal, terminal-blue, terminal-monochrome, tokyo-night, WAN

## custom (17 groups)

Hardware/vendor-branded presets applied (e.g., asusrog, razer, framework). Same mechanism as registry but themed to match specific hardware.

Groups: acer, alienware, aorus, dell, framework, hpspectre, mac, msi-creator, msi-dragon, omen, razer, ROG, surface, system76, thinkpad, yoga, zenbook

## generic (1 group)

Uses a fixed color name. No DMS preset is applied.

Groups: cute (color: coral)
